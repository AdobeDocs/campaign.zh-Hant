# 使用Web表單收集和更新配置檔案

使用「市場活動」可以輕鬆高效地建立Web表單並收集和管理配置檔案資料。 您可以在您的網站共用這些表單，讓您的聯絡人輕鬆提供資訊。 資料將發送到市場活動以建立或更新其配置檔案。
