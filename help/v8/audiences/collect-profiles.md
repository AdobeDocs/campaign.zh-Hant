# 使用網路表單收集和更新設定檔

使用Campaign建立網路表單，以及輕鬆、有效率地收集和管理設定檔資料。 您可以在您的網站共用這些表單，讓您的聯絡人輕鬆提供資訊。 資料會傳送至Campaign以建立或更新其設定檔。
