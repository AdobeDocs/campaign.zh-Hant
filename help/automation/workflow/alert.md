# 警報{#alert}



安 **警報** 活動向一組運算子發送消息。 它的操作方式與審批活動相同，但在本例中不預期有任何響應。

![](assets/edit_alerte.png)

警報不是持久的，因此從控制台中不可見。 為接收通知，所分配組的操作員必須擁有完整的電子郵件地址。 此活動的配置與 **批准**。 用於通知運算子的預設傳遞模板為「alertAssignee」。
