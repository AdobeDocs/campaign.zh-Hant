# 訪問外部資料庫(FDA){#accessing-an-external-database-fda}

Adobe Campaign 提供同盟資料存取 (FDA) 選項，以處理儲存在一或多個外部資料庫中的資訊：您不需要變更 Adobe Campaign 資料的結構就可以存取外部資料。

通過多個活動，您可以與外部資料庫中的資料交互：

* 查詢
* 分割
* 擴充
* 資料載入

FDA選項的使用取決於您的Adobe Campaign許可證。 詳細說明了配置此選項以及與使用此選項相關的最佳做法和限制 [此部分](https://experienceleague.adobe.com/docs/campaign/campaign-v8/connect/fda.html){target=&quot;_blank&quot;}。
