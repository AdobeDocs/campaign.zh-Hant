# 時間限制{#time-constraint}

A **時間約束** 活動可以延遲任務的執行或放棄任務。

輸入活動的標籤，並指定必須暫停工作流任務的時間範圍。

當 **[!UICONTROL Try again later if outside of execution period]** 的子菜單。 如果希望在工作流操作暫停後將其永久放棄，請取消選擇此選項。
