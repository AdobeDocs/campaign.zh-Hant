# 時間限制{#time-constraint}

A **時間限制** 活動可讓您延遲或捨棄任務的執行。

輸入活動的標籤，並指定工作流程任務必須暫停的時間範圍。

當 **[!UICONTROL Try again later if outside of execution period]** 選項時，可在執行時間範圍外重新啟動工作。 如果您希望工作流程動作在暫停後永久放棄，請取消選取此選項。
